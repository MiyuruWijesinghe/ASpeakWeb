ASpeak

A Mobile Application Approach to Translate Sri Lankan Sign Language to Text in Real-Time

Get Started

What is ASpeak?

ASpeak is an application that translates static and dynamic sign language gestures into text in real-time basis. It is a cost-effective and user-friendly solution as it has been developed in the form of a mobile application. This application has below components.

Letâ€™s Talk

This component can be used to communicate between normal people and aurally handicapped people as an interpreter.

Word To Sign

This component allows user to type a particular word and see the relevant gif generated on the screen.

Learning Corner

When the user is navigated to this screen, the mobile camera turns on and loads to the camera frame. When user presses "Start Detection" button, real-time object detection starts and the relevant output displays to the user.

Upload & Convert

This component allows user to upload either a dynamic sign or static sign. Then generate the prediction and display it to the user in the output section when the "Generate Prediction" button is pressed.

Features

Features

Usable for Communication

A helpful interpreter to communicate between normal people and aurally handicapped people.

Real-time Identification

Capable of interpreting over 150 Sri Lankan Sign Language gestures in real-time basis.

Identify Static Gestures

Capable of interpreting static Sri Lankan Sign Language gestures in real-time basis.

Identify Dynamic Gestures

Capable of interpreting dynamic Sri Lankan Sign Language gestures in real-time basis.

Helpful for Beginners

Learning corner feature is helpful for the beginners to learn gestures used in Sri Lankan Sign Language.
